# Anti Refresh X

Extension de Chrome que evita que X (antes Twitter) actualice la timeline automaticamente mientras estas leyendo.

### Instalacion (Modo Desarrollador)
1. Descarga y extrae `Anti-Refresh-X.zip` (o usa esta carpeta directamente).
2. Abre Chrome y entra a `chrome://extensions`.
3. Activa **Modo de desarrollador** (arriba a la derecha).
4. Haz clic en **Cargar descomprimida**.
5. Selecciona la carpeta de la extension que contiene `manifest.json`.

### Prueba rapida
1. Abre `https://x.com/home`.
2. Desplazate hacia abajo en la timeline.
3. Espera a que lleguen publicaciones nuevas.
4. El feed no debe saltar/actualizarse solo mientras estas leyendo.
5. El refresh manual (`F5`) sigue funcionando normalmente.
