# Anti Refresh X

Chrome extension that prevents X (formerly Twitter) from auto-refreshing the timeline while you are reading.

### Install (Developer Mode)
1. Download and extract `Anti-Refresh-X.zip` (or use this folder directly).
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode** (top-right).
4. Click **Load unpacked**.
5. Select the extension folder containing `manifest.json`.

### Quick test
1. Open `https://x.com/home`.
2. Scroll down in your timeline.
3. Wait for new posts to arrive.
4. The feed should not jump/refresh automatically while you are reading.
5. Manual refresh (`F5`) still works normally.
