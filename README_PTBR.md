# Anti Refresh X

Extensao para Chrome que impede o X (antigo Twitter) de atualizar a timeline automaticamente enquanto voce esta lendo.

### Instalacao (Modo Desenvolvedor)
1. Baixe e extraia `Anti-Refresh-X.zip` (ou use esta pasta direto).
2. Abra o Chrome e acesse `chrome://extensions`.
3. Ative o **Modo do desenvolvedor** (canto superior direito).
4. Clique em **Carregar sem compactacao**.
5. Selecione a pasta da extensao que contem o `manifest.json`.

### Teste rapido
1. Abra `https://x.com/home`.
2. Role a timeline para baixo.
3. Aguarde novos posts aparecerem.
4. O feed nao deve pular/atualizar sozinho enquanto voce esta lendo.
5. Refresh manual (`F5`) continua funcionando normalmente.
